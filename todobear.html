To Do Bear is a mobile to-do application for users who want/need to stay organized with various tasks. This application allows users to create different lists based on category or groups of people. They are able to add specific members to each list and assign tasks to one or more persons in the list. The multiple view modes available allow the user to easily scan different types of tasks, including by due date. The teddy bear theme attracts users who want an endearing UI while managing their tasks.

Iteratively design a mobile to-do application by going through the entire prototyping process, from low to high-fidelity. Begin by creating low-fidelity paper prototypes using wizard of oz techniques for user testing. Incorporate user feedback to create new iterations, and gradually increase the fidelity, including an interactive prototype using HTML, CSS, and JavaScript/jQuery.
